Software Engineer at ShortCircuit Inc. (YC W2013)

September 2012 - present

At ShortCircuit, I worked on the real-time mass transit directions app Rover. I was primarily responsible for the backend, which was mainly Python with Flask. Here are a few of the things I created for Rover:

    Algorithm for routing via mass transit

    The algorithm was implemented based on a recently published research paper. It was a replacement for our older routing algorithm, with a performance increase of up to two orders of magnitude and significant savings in memory. This prevented a backend rewrite from Python to a performance language.

    Walking directions for Open Street Maps

    Shortest path algorithm which provided a visual guide for our users and distance calculations for the transit routing algorithm. Due to the size of OSM data, I created a fast, dynamic loading scheme that prevented having to keep all the data in memory at once.

    Other Responsibilities

  • Designed and implemented majority of the current production backend from ground up.
  • Business logic for handling real-time data.
  • Backend production releases to EC2 and Heroku.
